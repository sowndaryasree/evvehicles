## Overview
This project analyzes an Electric Vehicle (EV) Fleet dataset using Python and Google Colab.

## Features
- Vehicle Running Count
- Manufacturer-wise Vehicle Count
- Manufacturer-wise Revenue Analysis
- Manufacturer-wise Maintenance Cost
- Month-wise Revenue Generation
- Driver Revenue Analysis
- Driver Violation Analysis
- Maintenance Cost per Vehicle

## Technologies Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Dataset
The dataset contains 10,000 records of EV fleet operations including:
- Vehicle Details
- Driver Details
- Revenue Information
- Maintenance Records
- Battery Information
- Violation Records
